# ASP.NET Core User Management System

A full-stack ASP.NET Core MVC application for managing users with a SQL Server database. The project includes a responsive frontend interface and supports CRUD operations such as adding, updating, deleting, and listing users.

## Technologies
- ASP.NET Core MVC
- C#
- MSSQL
- Entity Framework Core
- Bootstrap

## Features
- User listing
- Add new users
- Update user information
- Delete users
- Database integration
- Responsive UI
